Arch Linux + Hyprland

A comprehensive guide to building a signed, secure, and aesthetic dual-boot Tiling Window Manager environment.

01. Core System & Secure Boot

Focus: Establishing a signed, secure dual-boot environment with Windows using GRUB and TPM modules.

Grub & TPM Installation

sudo grub-install --target=x86_64-efi --efi-directory=/boot/efi --bootloader-id=EndeavourOS --modules="tpm" --disable-shim-lock
  • Secure Boot: Use sbctl to self-sign the kernel and GRUB binaries for complete system integrity.
  • Key Maintenance: Always initialize pacman keys before installing major packages.
sudo pacman-key --init
sudo pacman-key --populate archlinux

02. Essential Tooling & Packages

Base & AUR Helper

  • Base Development: sudo pacman -S base-devel git
  • AUR Helper (yay): The backbone of custom Arch builds.
git clone https://aur.archlinux.org/yay.git
cd yay && makepkg -si

Fixes & Locales

If you hit "externally managed environment" errors in Python, use pipx or install via AUR.

"If builds fail due to locale errors, ensure your export is set."

export LC_ALL=en_US.UTF-8

04. Audio & Communication

PipeWire Stack

High-fidelity routing with PipeWire and GUI patchbays.

sudo pacman -S pipewire pipewire-pulse wireplumber pwvucontrol qpwgraph

Discord/Vesktop (Wayland Fixes)

  1. Use Vesktop for superior screen sharing on Wayland.
  2. Use xdg-desktop-portal-hyprland (remove -wlr to avoid conflicts).
  3. Update your hyprland.conf:
exec-once = dbus-update-activation-environment --systemd WAYLAND_DISPLAY XDG_CURRENT_DESKTOP

06. Troubleshooting Reference

Issue Solution
AUR apps not appearing Run update-desktop-database ~/.local/share/applications or restart session.
Discord Screen Share Check systemctl --user status xdg-desktop-portal; ensure Hyprland portal is active.
Python Errors Use yay -S python-sphinx for build deps; export LC_ALL for locale.
Duplicate Discord Check both pacman and flatpak lists. Remove Flatpak to avoid path confusion.
RTKit / WirePlumber Add user to the realtime group or verify rtkit is installed.
Moving Windows In Hyprland, use Super + Shift + [1-9] to move to specific workspace.
Electron/Web Notifications Add ELECTRON_OZONE_PLATFORM_HINT=wayland to /etc/environment.
